麦田pt代码基于nexus ，于2010年由星星和独行虾改造完成，更适合学生用户使用，长期运行于纯ipv6环境下。

改造完成之后相继发布了0.1.1,0.1.2,0.2版本，修复了一些bug以及添加了一些功能。

此次发布的0.3版本是在许多人的共同维护下，修补bug，完善、添加功能之后整理出的更容易安装使用的版本。但是仍然有许多已知bug未能修复，需求的功能未能添加。

主要的缺点是多语言支持被破坏了，以及配置起来更麻烦。之后更新的代码由于精力和能力有限，更改的配置相关的代码直接写到了相应页面，输出也直接用简体中文方式写到了相应页面。

这使得代码的整改挪用变得十分困难，需要查看、修改每个页面才能至少把麦粒，麦田什么的相关词语改掉。对此表示很遗憾，嘛，好遗憾。

主要的改动有，增加偷麦粒应用，彩票应用，改名，购买促销应用，群聊自动喊话机器人
求种，回收站，自助求邀，标题自动生成系统，自动获取imdb和豆瓣链接系统
完善考核系统，添加二级分类图标，自助绑定学号，完善站内信，加入@功能，引用发布功能
等等，详细改动日志请查看麦田史册（更详细的会指向论坛
